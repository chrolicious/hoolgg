Notes:
All three options use the same interaction patterns from the Mario Wonder references:
— Dark backgrounds with subtle checkered texture overlay
— Bold primary color for active/selected states
— White/cream for secondary/inactive elements
— Thick bordered buttons (rounded pill shape)
— Generous border-radius on all surfaces

A = Closest to WoW's own visual language. Warm, prestigious, familiar to WoW players.
B = Modern, stands apart from WoW. Fresh, techy, high contrast. Could feel more "tool" than "game."
C = Two-accent system. Purple for ambient/surface, gold for primary actions. Most visually rich, but more complex to maintain.
